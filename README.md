# Plante de Constellation

Plante de Constellation est l’un des sketches de l’installation 
Plantes numériques dans le cadre de mon projet de DNSEP.  
Elle représente une plante virtuelle dont les éléments s’organisent 
comme des constellations, créant un motif vivant qui évolue sur les cercles au sol.

Visuel du code : https://editor.p5js.org/celinesurlalune/full/4Lk94Rz88
